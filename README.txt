TITLE
    AminoAcidMapper
    
DESCRIPTION
    This simple code will identify and map the amino acid, on a given peptide
    chain, that is been coded by a codon which contains the nucleotide at a given
    position on a given sequence.

DEPENDENCIES
    This module requires Python and BioPython Installed on you system.

PLATFORMS    
    Linux:
        Works Well
    
    Windows:
        Works Well
    
USGAE
    Check the example.py
